Cascading Style Sheets

Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web

O atributo background-image é usado para definir uma imagem de fundo para um elemento HTML. Os valores permitidos para este atributo são: Uma URL para uma imagem: esta é a forma mais comum de definir uma imagem de fundo. A URL deve apontar para uma imagem que esteja armazenada no servidor web. Uma cor: esta opção define uma cor sólida como imagem de fundo. A cor pode ser definida usando um valor hexadecimal, um nome de cor ou um código de cor CSS. Uma gradiente: esta opção define um gradiente como imagem de fundo. Um gradiente é uma transição suave entre duas ou mais cores.
